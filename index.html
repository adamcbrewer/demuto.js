demuto.js

Changing your styles for the worse!

This should be used to test progressive enhanced CSS during development. The quick-toggling of Modernizr class-names appended to the html tag can be done either by dragging this bookmarklet to your bookmarks toolbar, or by creating a new instance of the class provided in the demuto.js script.
